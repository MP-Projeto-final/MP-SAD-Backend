(constant) createDonationSchema

Joi schema for creating a donation.

This schema validates the data for creating a new donation.

Properties:
Name Type Attributes Description
descricao string

Description of the donation. Must be a string with a maximum length of 300 characters and is required.
destino_cep string

Destination ZIP code. Must be a string with exactly 8 characters and is required.
destino_rua string

Destination street. Must be a string with a maximum length of 255 characters and is required.
destino_numero string

Destination number. Must be a string with a maximum length of 50 characters and is required.
destino_complemento string <optional>

Destination complement. Optional field that can be a string with a maximum length of 255 characters or an empty string.
destino_bairro string

Destination neighborhood. Must be a string with a maximum length of 255 characters and is required.
destino_cidade string

Destination city. Must be a string with a maximum length of 255 characters and is required.
destino_estado string

Destination state. Must be a string with exactly 2 characters (usually an abbreviation) and is required.
